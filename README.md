## 🛠️ Tech Stack & Architecture

This project was developed using the most modern, resilient technologies in the contemporary web ecosystem:

* **[Next.js 15 (App Router)](https://nextjs.org/)**: Enterprise-grade React framework providing hybrid rendering (SSR / SSG), optimized dynamic routing, and near-instantaneous page loads.
* **[React 19 & TypeScript](https://react.dev/)**: Strict static typing that ensures fault tolerance, enhanced developer ergonomics, and highly cohesive, modular component design.
* **[Tailwind CSS](https://tailwindcss.com/)**: Reactive, utility-first styling powered by a custom Design System featuring executive color tokens, seamless **Dark / Light Mode** transitions, and modern glassmorphism aesthetics.
* **[Framer Motion](https://www.framer.com/motion/)**: Orchestrated UI animations, smooth 60fps layout transitions, scroll-triggered reveal components (`Reveal`), and engaging micro-interactions.
* **[Lucide Icons & shadcn-style UI primitives](https://lucide.dev/)**: Lightweight, consistent vector icons providing clean visual hierarchy across all screen densities.
